# Transaction Processing Microservice

A Spring Boot microservice that processes banking transactions including deposits, withdrawals, and transaction history retrieval.

## Features

- RESTful APIs for deposits and withdrawals
- Transaction history retrieval
- Layered architecture (Controller, Service, Repository)
- H2 in-memory database
- Spring Data JPA persistence
- Designed to simulate workloads of 5000+ transactions

## Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Maven
- H2 Database
